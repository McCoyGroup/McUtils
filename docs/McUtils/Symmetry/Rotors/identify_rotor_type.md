```python
identify_rotor_type(moms: numpy.ndarray, tol=1e-08, zero_tol=1e-06): 
```
**LLM Docstring**

Classify principal moments as atomic, linear, spherical, oblate, prolate, or asymmetric and independently test the planar moment relation.
  - `moms`: `np.ndarray`
    > Value used as `moms` by the implementation.
  - `tol`: `object`
    > Numerical tolerance used for geometric or equality tests. Defaults to `1e-08`.
  - `zero_tol`: `object`
    > Value used as `zero_tol` by the implementation. Defaults to `1e-06`.
  - `:returns`: `tuple[RotorTypes, bool]`
    > A pair of rotor type and planar flag.
