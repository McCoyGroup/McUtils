```python
point_group_data(key, n=None, prop=None, **etc): 
```
**LLM Docstring**

Look up fixed-size or parametrized point-group generators and optionally evaluate one requested property.
  - `key`: `object`
    > Point-group family key or fixed group name.
  - `n`: `object`
    > Group order or problem size used to construct the requested representation. Defaults to `None`.
  - `prop`: `object`
    > Value used as `prop` by the implementation. Defaults to `None`.
  - `etc`: `dict`
    > Additional keyword options forwarded to the selected constructor, generator, or plotting backend.
  - `:returns`: `dict | object | None`
    > The group-data mapping, a generated property, or `None` for an unavailable property.
