```python
symmetric_group_character_table(n, tableaux=None, partitions=None, return_partitions=False, return_weights=False): 
```
**LLM Docstring**

Construct the symmetric-group character table with the Murnaghan–Nakayama rule over standard Young tableaux.
  - `n`: `object`
    > Group order or problem size used to construct the requested representation.
  - `tableaux`: `object`
    > Standard Young tableaux used for the Murnaghan–Nakayama construction. Defaults to `None`.
  - `partitions`: `object`
    > Integer partitions that define conjugacy classes. Defaults to `None`.
  - `return_partitions`: `object`
    > Value used as `return_partitions` by the implementation. Defaults to `False`.
  - `return_weights`: `object`
    > Value used as `return_weights` by the implementation. Defaults to `False`.
  - `:returns`: `np.ndarray | tuple`
    > The character table, optionally followed by partitions and class weights.
