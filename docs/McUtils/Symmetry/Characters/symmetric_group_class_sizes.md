```python
symmetric_group_class_sizes(n, partitions=None): 
```
**LLM Docstring**

Compute conjugacy-class sizes for every supplied partition of `S_n`.
  - `n`: `object`
    > Group order or problem size used to construct the requested representation.
  - `partitions`: `object`
    > Integer partitions that define conjugacy classes. Defaults to `None`.
  - `:returns`: `np.ndarray`
    > An array of class sizes in partition order.
