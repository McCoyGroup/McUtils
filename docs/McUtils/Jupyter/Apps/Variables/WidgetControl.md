when one isn't given.

    > the variable (name or synchronizer)
  - `control_type`: `Any`
    > the widget type (inferred from the settings if omitted)
  - `widget`: `Any`
    > an explicit widget (built if omitted)
  - `settings`: `Any`
    > the widget settings
