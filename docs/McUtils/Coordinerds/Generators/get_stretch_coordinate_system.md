```python
get_stretch_coordinate_system(stretches, include_bends=True, include_dihedrals=True): 
```
**LLM Docstring**

Construct bends and dihedrals implied by a set of stretches.

Bends are generated from shared-atom stretch pairs whenever either bends or dihedrals are requested. Dihedrals are then generated by joining the resulting bends. Disabled coordinate classes are represented by `None`, while the original `stretches` object is returned unchanged.
  - `stretches`: `collections.abc.Sequence[tuple[int, int]]`
    > Bond coordinates used as the connectivity graph.
  - `include_bends`: `bool`
    > Whether to include the generated angle list in the result.
  - `include_dihedrals`: `bool`
    > Whether to generate and include dihedral coordinates.
  - `:returns`: `tuple`
    > `(stretches, angles, dihedrals)`, with disabled components set to `None`.
