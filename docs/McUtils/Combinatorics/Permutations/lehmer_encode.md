```python
lehmer_encode(perms, dtype=None): 
```
**LLM Docstring**

Encode permutations as their Lehmer-code integers (a factorial-base ranking),
choosing an integer or object dtype large enough for the factorials.
  - `perms`: `np.ndarray`
    > the permutations, shape `(..., ndim)`
  - `dtype`: `Any`
    > the output dtype (inferred if omitted)
  - `:returns`: `np.ndarray`
    > the Lehmer codes, shape `(...,)`
