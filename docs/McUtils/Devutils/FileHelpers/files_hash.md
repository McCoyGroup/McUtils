```python
files_hash(files, base=None, bits=None, id_generator=None): 
```
**LLM Docstring**

Hash the concatenated contents of several files into a single digest.
  - `files`: `Iterable[str]`
    > the file paths
  - `base`: `int | None`
    > the string-encoding base
  - `bits`: `int | None`
    > truncate the digest to this many bytes
  - `id_generator`: `Any`
    > the id encoding
  - `:returns`: `_`
    > the combined hash
