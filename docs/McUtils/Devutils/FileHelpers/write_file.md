```python
write_file(file, data, mode='w+', **opts): 
```
**LLM Docstring**

Write data to a file (path or stream).
  - `file`: `Any`
    > the file path or stream
  - `data`: `Any`
    > the data to write
  - `mode`: `str`
    > the open mode
  - `opts`: `Any`
    > options forwarded to `open`
  - `:returns`: `_`
    > the file
