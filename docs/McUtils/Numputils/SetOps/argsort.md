```python
argsort(ar): 
```
**LLM Docstring**

Stable argsort that supports multi-dimensional rows by coercing them to a
compound dtype first.

Uses a mergesort for stability and down-casts the resulting permutation to a
minimal dtype.
  - `ar`: `np.ndarray`
    > the array to sort
  - `:returns`: `np.ndarray`
    > the (down-cast) sort permutation
