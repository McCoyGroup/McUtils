```python
is_numeric(obj, types=None): 
```
**LLM Docstring**

Test whether an object is a scalar numeric value (int or float, or a
zero-dimensional numeric array).
  - `obj`: `Any`
    > the object to test
  - `types`: `tuple | None`
    > the numeric types to accept (defaults to int/float types)
  - `:returns`: `bool`
    > whether the object is a numeric scalar
